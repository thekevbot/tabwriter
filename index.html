=============================
=== WELCOME TO TAB WRITER ===
=============================

Just start typing.

=== Keyboard Shortcuts ===
==========================

1. [CMD + SHIFT + '] - Enter Command mode
2. [CMD + SHIFT + RETURN] - Insert marker
3. [CMD + SHIFT + <] - Previous marker
4. [CMD + SHIFT + >] - Next marker

=== Useful Commands ===
=======================

1. HELP - Insert help text
2. GOTO {DDMMYY} - Jump to marker
